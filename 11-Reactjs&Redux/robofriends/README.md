# robofriends

This React project is the part of the [Udemy](https://www.udemy.com/course/the-complete-web-developer-zero-to-mastery/) course.

To run the project:

1. Clone this repo
2. Run `npm install`
3. Run `npm start`

If you have a issue to install the project, re-run the `npm install` by adding `sudo`
